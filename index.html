BRAIN DISEASE DIAGNOSIS USING MACHINE LEARNING

This project improved brain tumor segmentation in MRI images, addressing challenges like tissue variation and noise. We applied multimodal image segmentation, Gabor filter-based feature extraction, and spatially guided generative models for accurate tumor boundary detection. Additionally, the Bat Algorithm was used to optimize the segmentation process, enhancing accuracy. This approach strengthened understanding of medical image analysis, machine learning, and optimization, and helped demonstrate how advanced techniques can significantly improve the precision and reliability of brain tumor identification in clinical imaging.

Key Technologies

Python, Gabor Filters, Bat Algorithm, Machine Learning, Image Segmentation, Generative Models, MRI Brain Imaging, Feature Extraction

LUNG DISEAES PREDICTION USING DEEP LEARNING

This project involved developing two deep learning models using pre-trained Convolutional Neural Networks (CNN) — VGG16 and VGG19 — for accurate pneumonia detection using the Kaggle chest X-ray image dataset. The models achieved impressive accuracy rates of 96.26% and 95.32%, respectively. The project demonstrates the effectiveness of transfer learning for medical image classification, while also highlighting that accuracy can further improve when trained on larger and more diverse datasets. This project sharpened skills in data preprocessing, model tuning, and performance evaluation using real-world medical datasets.

Key Technologies

Python, TensorFlow / Keras, CNN (Convolutional Neural Network), VGG16, VGG19, Kaggle Dataset, Image Classification, Transfer Learning
